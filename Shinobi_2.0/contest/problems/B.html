Problem B — Increasing Array

Time Limit: 1.00 s
Memory Limit: 512 MB

You are given an array of n integers. You want to modify the array so that it is increasing, i.e., every element is at least as large as the previous element.

On each move, you may increase the value of any element by exactly one. What is the minimum number of moves required?

Input

The first line contains an integer n: the size of the array.

The second line contains n integers x1, x2, ..., xn: the contents of the array.

Output

Print one integer: the minimum number of moves required.

Constraints

  • 1 ≤ n ≤ 2 · 105
  • 1 ≤ xi ≤ 109

Example

Input 
5
3 2 5 1 7
Output 
5
Explanation:
The initial array is [3, 2, 5, 1, 7].
- Increase the 2nd element by 1 to make it 3. (Array becomes [3, 3, 5, 1, 7]. Cost = 1 move).
- Increase the 4th element by 4 to make it 5. (Array becomes [3, 3, 5, 5, 7]. Cost = 4 moves).
Total minimum moves = 1 + 4 = 5.
