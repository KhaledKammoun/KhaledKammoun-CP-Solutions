Problem C — Operation Carthage: Signal Intercept

Time Limit: 2.00 s
Memory Limit: 256 MB
World War 3 has engulfed the globe. Because of Tunisia's critical strategic position bridging the Mediterranean, Europe, and Africa, the country's airspace has become a massive theater for electronic warfare. The Tunisian Cyber Defense Command has intercepted a raw data stream containing N base radar frequencies used by enemy stealth fleets.

The enemy's communication protocol encrypts their strike coordinates by combining pairs of these base frequencies. Specifically, a valid target coordinate is formed by adding the values of any two base frequencies transmitted at distinct temporal indices in the array.

To calibrate the national jamming grid and protect the coastline, the defense engineers at ISIMS need a complete map of the attack matrix. You must find all possible combined frequencies (sums) that can be formed by pairing distinct indices, and determine exactly how many ways each sum can be constructed.

Two pairs of indices (i, j) and (x, y) are considered identical if they contain the exact same set of indices. Because N is extremely large, a brute-force approach to checking all pairs will not be fast enough to jam the signals in time.

Input

The first line contains a single integer N — the number of intercepted base frequencies.

The second line contains N space-separated integers A1, A2, ..., AN — the values of the base frequencies.

Output

Print multiple lines representing the threat matrix. For every combined frequency S that can be formed by at least one valid pair, print two space-separated integers: the combined frequency S and the number of distinct index pairs (i, j) (where i < j) such that Ai + Aj = S.

The combined frequencies must be printed in strictly increasing order. Do not print frequencies that have a formation count of 0.

Constraints

  • 2 ≤ N ≤ 200,000
  • 0 ≤ Ai ≤ 100,000

Example

Input 
5
1 2 3 1 2
Output 
2 1
3 4
4 3
5 2

Note

Let's simulate the combinations for the intercepted frequencies [1, 2, 3, 1, 2]:

  • Frequency 2: Can be formed by indices (0, 3) → 1 + 1. (1 way)
  • Frequency 3: Can be formed by indices (0, 1), (0, 4), (1, 3), (3, 4) → 1 + 2, 1 + 2, 2 + 1, 1 + 2. (4 ways)
  • Frequency 4: Can be formed by indices (0, 2), (1, 4), (2, 3) → 1 + 3, 2 + 2, 3 + 1. (3 ways)
  • Frequency 5: Can be formed by indices (1, 2), (2, 4) → 2 + 3, 3 + 2. (2 ways)
